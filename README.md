## 第三步：登录 KiwiVM 控制面板

KiwiVM 是搬瓦工自己开发的 VPS 管理后台。**所有关于这台服务器的操作——开关机、重装系统、换机房、看流量、改密码——都在这里完成。**

进入方法：

1. 登录搬瓦工账号，进 **Client Area → Services → My Services**
2. 找到你买的 VPS，点旁边的 **KiwiVM Control Panel** 按钮
3. 自动跳转进入面板

第一次进去看着有点乱，但其实你用得最多的就那几个地方：

- **Main Controls**：看 IP 地址、SSH 端口、流量使用情况，以及开机/关机/重启按钮
- **Root password modification**：生成或重置 root 密码。**必须先关机才能操作**，否则会失败
- **Install new OS**：重装系统，选 Ubuntu 或 Debian 都行，装之前也要先关机
- **Migrate to another DC**：换机房，不损失数据，IP 会变

## 第四步：拿到三样东西，准备连 SSH

登录 VPS 需要三个信息，缺一不可：

1. **IP 地址** — KiwiVM 首页 Main Controls 里直接看到
2. **SSH 端口** — 同一个页面，就在 IP 下面，不是 22，是个随机数字，比如 `28865`
3. **root 密码** — 去 KiwiVM 左侧菜单 → Root password modification → 先关机 → Generate and set new root password → 把生成的密码复制下来。只显示一次，存好。

这三样齐了，可以连了。

## 第五步：用 SSH 工具连上去

### Windows 用户

推荐用 **Termius**，界面干净，免费够用，跨平台还能同步配置。安装好之后：

1. 新建连接，填入 IP 地址
2. Port 那里填上你的 SSH 端口（不是 22）
3. Username 填 `root`
4. Password 填刚才保存的密码
5. 连接

连上去会看到一行命令提示符，类似 `root@bandwagonhost:~#`。到这一步就成功了。

### Mac / Linux 用户

直接开终端，命令格式是：

bash
ssh root@你的IP -p 你的SSH端口


举个例子，如果 IP 是 `123.45.67.89`，端口是 `28865`：

bash
ssh root@123.45.67.89 -p 28865


然后输入密码，回车。

## 常见问题

**Q：搬瓦工 VPS 购买后要等多久才能用？**

A：基本是即时开通，支付成功后几分钟内就收到邮件，马上可以连 SSH。

**Q：SSH 连不上，一直超时怎么办？**

A：先确认端口号是否用了 KiwiVM 里显示的随机端口，不是默认 22。其次看 KiwiVM 里 VPS 是否处于 Running 状态。如果 IP 被封，需要在 KiwiVM 里换机房（IP 会变）或者付费换 IP。

**Q：系统重装会不会把所有数据清掉？**

A：会。重装系统是完整抹盘重装，所有文件都会消失。操作前先在 KiwiVM 里做一个快照（Snapshot），数据安全有保障。

**Q：每个月流量用完了怎么办？**

A：流量清零是按账单周期来的，到期自动重置。如果提前用完，VPS 会降速或暂停对外服务，可以在 KiwiVM 里买额外流量，也可以等下个周期。

**Q：搬瓦工支持支付宝付款吗？**

A：支持。结账页面选支付宝即可，不需要信用卡，国内用户购买很方便。
